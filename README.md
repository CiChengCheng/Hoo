# Hoo

🚀 Hoo是一个利用`Android Jetpack`架构组件编写的`MVVM模式`的入门级App，目的是**用于指导新手使用`Android Jetpack`**。

### 2. 除了Android Jetpack

相信除了`Android jetpack`架构组件的知识以外，你还可以学到：

- `ConstraintLayout`的使用，几乎每个界面布局都使用了`ConstraintLayout`。
- 符合《阿里Android手册》的标准化资源文件命名方式。
- `Kotlin`
- ...
